# Adventure Works Product Performance Dashboard

Actionable Insights Over Aesthetics — Power BI Dashboard built with Maven Analytics Adventure Works dataset.

![Dashboard Overview](main/IMG_2601.png)

## 🎯 The Mission

Too many analysts focus on making dashboards look beautiful while forgetting the real purpose:

> Delivering clear, actionable insights that help businesses make better decisions.

This project was built with that principle in mind. Every visual is designed to answer real business questions instead of just looking good.

## 📊 Key Insights

- Overall Performance: $24.9M Total Revenue | $10.5M Profit | 25.2K Orders | 2.17% Return Rate
- Accessories dominate with 17K orders, making it the strongest category
- Sport-100 Helmets (Blue, Red, Black) are top performers but some are missing monthly profit targets
- "Shorts" is the most returned product type — clear red flag for quality, sizing, or pricing review
- Strong revenue growth trend since mid-2021 with visible monthly fluctuations
- Interactive product drill-through reveals orders, revenue, and profit vs. targets + return patterns (example: Sport-100 Helmet, Blue)
- Geographic view showing performance concentration in North America, Europe, and Pacific regions

This dashboard turns data into a clear pathway for action — what to push harder, what to fix, and where to investigate further.
